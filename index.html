All Rise

You are the judge. Five sources stand before your court. Cross-examine each one, then rule: Admissible or Dismissed.

How to Play

  1. Review the Exhibit: Read the source metadata and excerpt carefully. Hints like "(Peer-Reviewed)" have been removed, so you must dig into the text!
  2. Cross-Examine: Answer the four evaluation questions. If you aren't certain, select "MAYBE".
  3. Rule: Decide if the source is Admissible (scholarly, credible) or Dismissed (biased, unreliable).
